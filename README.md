# buscaminas.py
Clásico juego de Buscaminas desarrollado en Python, con foco en la lógica del juego, la corrección del comportamiento y el testing automatizado. El proyecto fue realizado con fines académicos, priorizando buenas técnicas practicas de programación y código testeable.

Objetivos del proyecto:
Implementar la lógica completa del juego Buscaminas, separar correctamente responsabilidades (estado, reglas, acciones), garantizar la corrección del código mediante tests automatizados, alcanzar una alta cobertura de tests sobre líneas y ramas.

Tecnologías utilizadas:
Python, unittest (testing automatizado), coverage (medición de cobertura), control de versiones con Git.

Funcionalidades principales:
Creación de tableros con minas, descubrimiento de celdas, marcado y desmarcado de celdas, detección de victoria y derrota, guardado y carga del estado del juego, validación de movimientos y estados inválidos.

El proyecto cuenta con una suite de tests automatizados que valida:
Comportamiento correcto de las reglas del juego, casos borde y escenarios inválidos, persistencia del estado.

Cobertura de tests:
Más del 95% de cobertura en líneas y ramas de ejecución.

Conceptos aplicados:
Programación estructurada, programación orientada a objetos, separación de responsabilidades, desarrollo basado en tests, diseño de código mantenible.

Estado del proyecto:
Proyecto finalizado en el marco académico, con posibilidad de extensión (interfaz gráfica, mejoras de arquitectura, nuevas reglas).
